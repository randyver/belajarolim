Keterbagian

Untuk setiap bilangan bulat \(m\) dan bilangan bulat taknol \(n\), dapat dikatakan \(m\) habis dibagi \(n\) atau \(n\) membagi \(m\) jika terdapat bilangan bulat \(k\) sehingga \(m=kn\), dengan kata lain \(\frac{m}{n}\) adalah bilangan bulat. Melambangkannya dengan \(n|m\), sebaliknya jika \(n\) tidak habis membagi \(m\), maka \(n\nmid m\).

Properti Keterbagian

  1. \(x|x\) (reflekstif)

  2. jika \(x|y\) dan \(y|z\), maka \(x|z\) (transitiv)

  3. jika \(x|y\) dan \(y\neq 0\), maka \(|x| \leq |y|\)

  4. jika \(x|y\) dan \(x|z\), maka \(x|\alpha y+\beta z\) untuk setiap bilangan bulat \(\alpha\) dan \(\beta\)

  5. Jika \(x|y\) dan \(x|y\pm z\), maka \(x|z\)

  6. jika \(x|y\) dan \(y|x\), maka \(|x|=|y|\)

  7. jika \(x|y\) dan \(y\neq 0\), maka \(\frac{y}{x} |y\)

  8. untuk \(z\neq 0\), \(x|y\) jika dan hanya jika \(xz|yz\)

Preposisi

  1. Untuk \(n\) bilangan asli, maka \(a-b |a^n-b^n\)
    Bukti : Tinjau pemfaktoran dari \(a^n-b^n=(a-b)(a^{n-1}+a^{n-2}b+\dots+ab^{n-2}+b^{n-1})\). Jelas bahwa \(a^{n-1}+a^{n-2}b+\dots+ab^{n-2}+b^{n-1}\) bilangan bulat, sehingga sesuai definisi diperoleh \(a-b|a^n-b^n\).

  2. Untuk \(n\) bilangan ganjil, maka \(a+b|a^n+b^n\)
    Bukti : Khusu \(n\) bilangan ganjil, \(a^n+b^n=(a+b)(a^{n-1}-a^{n-2}b+\dots-ab^{n-2}+b^{n-1})\). Jelas bahwa \(a^{n-1}-a^{n-2}b+\dots-ab^{n-2}+b^{n-1})\) bilangan bulat, sehingga sesuai definisi diperoleh \(a+b|a^n+b^n\).

Paritas

  1. bilangan ganjil berbentuk \(2k+1\) untuk setiap bilangan bulat \(k\)

  2. bilangan genap berbentuk \(2m\) untuk setiap bilangan bulat \(m\)

  3. Jumlah dari dua bilangan ganjil adalah bilangan genap

  4. jumlah dari dua bilangan genap adalah bilangan genap

  5. jumlah dari bilangan ganjil dan bilangan genap adalah bilangan ganjil

  6. perkalian dari dua bilangan ganjil adalah bilangan ganjil

  7. perkalian dari beberapa bilangan bulat adalah genap jika dan hanya jika setidaknya satu bilangan genap

Aturan Keterbagian

Bilangan bulat positif \(n\) habis dibagi

  • \(2\), jika digit terakhirnya bilangan genap

  • \(3\), jika jumlah digitnya kelipatan \(3\)

  • \(4\), jika dua digit terakhirnya kelipatan \(4\)

  • \(5\), jika digit terakhirnya \(0\) atau \(5\)

  • \(6\), jika habis dibagi \(2\) dan \(3\)

  • \(7\), jika hasil pengurangan selain digit terakhir dengan dua kali digit terakhir adalah kelipatan \(7\)

  • \(8\), jika tiga digit terakhirnya kelipatan \(8\)

  • \(9\), jika jumlah digitnya kelipatan \(9\)

Modulo

Diberikan bilangan bulat \(a,b\) dan \(m\) dengan \(m \neq 0\). Bilangan \(a\) dan \(b\) dikatakan kongruen modulo \(m\) jika \(m\) membagi \(a-b\), dinotasikan dengan \(a \equiv b \pmod{m}\). Jika \(m\) tidak membagi \(a-b\), maka bilangan \(a\) dan \(b\) dikatakan tidak kongruen modulo \(m\) dan dinotasikan \(a \not\equiv b \pmod{m}\). Relasi "\(\equiv\)" pada definisi tersebut dinamakan relasi kongruensi. Beberapa karakteristik dasar terkait dengan modulo diberikan sebagai berikut. Diberikan bilangan bulat \(a,b,c,d\) dan \(m\).

  1. \(a \equiv a \pmod{m}\).

  2. Jika \(a \equiv b \pmod{m}\) dan \(b \equiv c \pmod{m}\), maka \(a \equiv c \pmod{m}\).

  3. \(a \equiv b \pmod{m}\), maka \(b \equiv a \pmod{m}\).

  4. Jika \(a \equiv b \pmod{m}\) dan \(c \equiv d \pmod{m}\), maka \(a+c \equiv b+d \pmod{m}\) dan \(a-c \equiv b-d \pmod{m}\).

  5. Jika \(a \equiv b \pmod{m}\), maka untuk setiap bilangan bulat \(k\) berlaku \(ka \equiv kb \pmod{m}\).

  6. Jika \(a \equiv b \pmod{m}\) dan \(c \equiv d \pmod{m}\), maka \(ac \equiv bd \pmod{m}\). Secara umum, jika \(a_i \equiv b_i \pmod{m}\), \(i=1,...,k\), maka \(a_1...a_k \equiv b_1...b_k \pmod{m}\). Lebih lanjut, jika \(a \equiv b \pmod{m}\), maka untuk setiap bilangan bulat positif \(k\) berlaku \(a^k \equiv b^k \pmod{m}\).

  7. \(a \equiv b \pmod{m_i}\), \(i=1,...,k\) jika dan hanya jika \[a \equiv b \pmod{lcm(m_1,...,m_k)}\] Secara khusus, jika \(m_1,...,m_k\) sepasang-sepasang relatif prima, maka \(a \equiv b \pmod{m_i}\), \(i=1,...,k\) jika dan hanya jika \(a \equiv b \pmod{m_1...m_k}\).

Quadratic Residue

Untuk setiap bilangan asli \(n\), maka

  • \(n^2 \equiv 0,1 \pmod{3}\)

  • \(n^2 \equiv 0,1 \pmod{4}\)

  • \(n^2 \equiv 0,1,4 \pmod{8}\)

  • \(n^3 \equiv 0,\pm 1 \pmod{9}\)

  • \(n^4 \equiv 0,1 \pmod{16}\)
